# テトリス学習プロジェクト - 100問の穴埋めチャレンジ

このプロジェクトでは、テトリスゲームを作りながらReactとJavaScriptのプログラミングを学習できます。

## 📚 学習の進め方

### 学習レベル

1. **Level 1（問1-20）**: 基本的な定数と変数
2. **Level 2（問21-40）**: テトロミノと配列操作
3. **Level 3（問41-60）**: 衝突検出とボード操作
4. **Level 4（問61-80）**: ゲームロジックと状態管理
5. **Level 5（問81-100）**: 高度な機能とReact Hooks

## 🎯 学習目標

- JavaScriptの基本構文を理解する
- 配列とオブジェクトの操作を習得する
- React Hooksの使い方を学ぶ
- ゲームロジックの実装方法を理解する
- 状態管理の概念を習得する

## 📝 ファイル構成

```
learning/
├── README.md                    # このファイル
├── QUESTIONS.md                 # 全100問の問題リストと解答
├── HINTS.md                     # 各問題のヒント集
├── templates/                   # 穴埋め用のテンプレートファイル
│   ├── level1-tetrominos.js    # Level 1: 基礎
│   ├── level2-game-logic.js    # Level 2: ゲームロジック
│   ├── level3-hooks.js         # Level 3: React Hooks
│   ├── level4-components.jsx   # Level 4: コンポーネント
│   └── level5-backend.js       # Level 5: バックエンド
└── answers/                     # 解答ファイル（完成版コード）
```

## 🚀 始め方

### ステップ1: 問題を読む
`QUESTIONS.md` を開いて、問1から順番に問題を確認してください。

### ステップ2: コードを書く
`templates/` フォルダ内の該当ファイルを開き、`/* 問X: ここに答えを書く */` の部分にコードを書いてください。

### ステップ3: テストする
各レベルのファイルを完成版のプロジェクトに統合してテストできます。

### ステップ4: 解答を確認
詰まったら `QUESTIONS.md` の解答セクション、または `HINTS.md` を参照してください。

## 💡 学習のコツ

1. **順番に進める**: 問題は段階的に難しくなるように設計されています
2. **すぐに答えを見ない**: まず自分で考えてみましょう
3. **ヒントを活用**: 完全に詰まったら `HINTS.md` を見てください
4. **動かして確認**: コードを書いたら実際に動かして確認しましょう
5. **理解を深める**: なぜそのコードが必要なのか考えながら進めましょう

## 📊 進捗管理

各レベルを完了したらチェックしましょう：

- [ ] Level 4（問61-80）: ゲームロジックと状態管理
